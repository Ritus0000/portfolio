# Dzianis Rytus Portfolio

Portfolio repository for selected work across AI, synthetic visual data, 3D simulation, rendering, architecture, and technical product systems.

This is intended to become the central portfolio hub: not one narrow project, but a structured collection of case studies showing how I think, design, build, and explain complex visual-technical systems.

## Core Direction

I work at the intersection of:

- synthetic 3D data generation for computer vision
- 3D simulation and rendering pipelines
- technical art / materials / environments
- AI systems and automation tools
- scientific and biomedical AI
- architecture, spatial design, and visual storytelling

The common thread is visual systems: building scenes, data, interfaces, and explanations that connect technical structure with strong visual clarity.

## Confidentiality Policy

Some real work is connected to university or laboratory projects and cannot be published directly. Public case studies in this repository must use one of these forms:

- public self-created demo assets
- simplified reproductions of workflow mechanics
- high-level diagrams without internal parameters
- sanitized screenshots or visuals only when explicitly approved
- original portfolio text written specifically for public use

See [docs/confidentiality.md](docs/confidentiality.md).
